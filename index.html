ABOUT

About Me

I work on autonomous robotics systems with a focus on ROS 2 navigation, robot simulation, embedded systems, and distributed robotic architectures. My work combines simulation workflows with real-world hardware deployment, particularly for resource-constrained robotic platforms.

My technical interests include autonomous mobile robot navigation, local planner benchmarking, decentralized swarm coordination, digital twins, embedded communication systems, and physics-based simulation environments using Gazebo Harmonic and PyBullet.

I currently serve as the Student Chair of the IEEE Robotics and Automation Society Student Branch at Christ University, where I organize robotics workshops, technical training programs, guest lectures, and hands-on engineering sessions focused on robotics, simulation, and autonomous systems development.

PROJECTS

Research and Engineering Work

ROS 2 / Navigation

Autonomous Mobile Robot Navigation System

Developed a ROS 2 Jazzy and Gazebo Harmonic based autonomous mobile robot platform using Nvidia Jetson Orin Nano for localization, path planning, obstacle avoidance, and autonomous indoor navigation.

  • Benchmarked DWB, MPPI, and RPP local planners within the Nav2 framework under structured indoor environments
  • Evaluated navigation performance using metrics including path efficiency, obstacle clearance, heading smoothness, and ROS process CPU utilization
  • Implemented differential-drive navigation workflows integrating SLAM, LiDAR-based obstacle detection, and autonomous waypoint following
  • Validated planner behavior across simulation and hardware deployments for resource-constrained robotic systems
Swarm Robotics

Collaborative Swarm Robotics System

Designed a decentralized swarm robotics framework for multi-agent coordination using MQTT-based communication and distributed task allocation workflows.

  • Implemented MQTT-based communication pipelines for inter-robot coordination and distributed state sharing
  • Developed decentralized task allocation workflows for collaborative robotic behavior
  • Integrated PID-based speed control and local motion coordination strategies
  • Explored scalable communication architectures for distributed robotic systems and autonomous swarm interaction
